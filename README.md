## 📋 Contenu

La mindmap couvre les thèmes suivants :

| Section | Contenu |
|---|---|
| 💼 Pérennité | Enjeux économiques, juridiques, financiers |
| 📋 Définitions | RGPD (EU 2016/679), Cybersécurité |
| 🏢 Étages conformité | DPO, Analyse risques, Priorisation, Gestion |
| ⚠️ Gestion violations | Notification CNIL 72h, Documentation, Plainte pénale |
| ✅ Bonnes pratiques | Traçabilité, Logs, Audits, SIEM |

## ✨ Fonctionnalités

- 🌳 **Arborescence interactive** — Cliquez pour développer/réduire chaque nœud
- 💻 **Exemples de code** — Blocs JavaScript illustrant chaque concept
- 🎬 **Vidéos intégrées** — YouTube embarqué pour chaque point
- 📊 **Exemples concrets** — Cas réels (Google, Amazon, WannaCry…)
- 🔍 **Panneau latéral** — Détails enrichis au clic sur chaque item
- 📱 **Responsive** — Adapté mobile et desktop

## 🛠️ Technologies

- **HTML5 / CSS3 / JavaScript** — Vanilla, aucune dépendance backend
- **Tailwind CSS** — Via CDN
- **Google Fonts** — Orbitron + Space Grotesk
- **YouTube IFrame API** — Vidéos embarquées

## 🖥️ Utilisation locale

Aucune installation requise. Ouvrez simplement le fichier dans un navigateur :

```bash
git clone https://github.com/maxlo245/mindmap-SLAM.git
cd mindmap-SLAM
# Ouvrir index.html ou rgpd-mindmap.html dans votre navigateur
```

> ⚠️ Pour les vidéos YouTube embarquées, une connexion Internet est requise.
